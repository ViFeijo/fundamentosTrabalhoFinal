## Descrição:
SharkHub é um programa que tem multifunções, ele cadastra, adiciona, consulta e exibe investidores, startups e pitches. Além disso, consegue calcular quanto dinheiro uma startup fez, apresenta o ranking que cada startup ficou e mostra o crescimento das startups que considera o percentual de crescimento selecionada pelo usuário durante 5 anos.

## Desafios:
1. Criar objetos infinitamente;
2. Identificar valuation pelo ID da startup;
3. Incrementar valores do desafio extra;
4. Tamanho do código;
5. Organização das contribuições;
## Soluções
1. Usando um método similar ao de adicionar um item a um vetor cheio, podemos criar objetos que não tem nome, assim permitindo que sejam criados infinitamente.
2. Chamando todos os objetos desejados por ID, podemos usar iterações e métodos .getID para chamar o que precisávamos.
3. Usando a recursão ensinada na última aula, resolvemos o problema, fazendo um método que calcula juros compostos, que utilizamos para calcular o crescimento da startup em uma determinada quantidade de anos.
4. Criando métodos com nomes reconhecíveis, modularizando a maioria do código e tendo convenções para nomes de variáveis, minimizamos o problema de ter um código muito longo e confuso. Ainda assim, tivemos um pouco de dificuldade com a quantidade de métodos e ter que scrollar até la embaixo.
5. Utilizamos um GitHub para controlar os arquivos e modificações de cada um dos membros do grupo.
