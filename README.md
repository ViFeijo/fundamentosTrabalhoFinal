## Descrição:
SharkHub é uma plataforma desenvolvida por Angelo, Pedro e Vitor que tem como objetivo proporcionar um banco de dados para startups e investidores, dando acesso fácil aos registros de investimentos. Além disso, o programa apresenta diferentes cálculos, como a valuation ou potencial de unicornização.
